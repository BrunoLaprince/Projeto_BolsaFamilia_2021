### Descrição
<p>Este projeto tem como objetivo realizar a análise de dados em formato CSV utilizando o Apache Spark, permitindo o processamento eficiente de grandes volumes de dados referente ao Bolsa Familia.<p>

---

### Pré-requisitos
- Python 3
- Apache Spark ou Databricks

---

### Estrutura de Dados (Execução)

├── dados/          # Arquivos CSV (não versionados)

├── src/            # Scripts de processamento com Spark

├── README.md       # Documentação do projeto

1. Criar a pasta `dados`
2. Colocar o arquivo CSV dentro dela
3. Executar os scripts da pasta `src`

---

### Observação
O arquivo CSV não é versionado no Git (provavelmente está no .gitignore).

Certifique-se de que o caminho do arquivo está correto dentro dos scripts.

Recomenda-se utilizar ambientes virtuais para gerenciar dependências.
